# Project Description

This repository contains a rapid public-health risk stratification framework for WASH recovery and health-system disruption using public humanitarian, displacement, environmental-health, and geospatial sources. The work is currently presentation-complete and is a candidate for conversion into a concise reproducible methods note.

The project is designed as a secondary-data assessment. Its claims should remain tied to dated public sources, and the repository should maintain clear separation between indexed inputs, interpretive prioritization, and limitations.

## Short description

Gaza WASH and health-system disruption assessment using public humanitarian, displacement, environmental-health, and geospatial sources.

## Scope boundary

Use as a public-health methods and risk-prioritization workspace. Do not imply field verification or causal attribution beyond the secondary-data design.
