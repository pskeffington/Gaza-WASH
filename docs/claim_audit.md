## Claims not yet permitted

- Individual-level disease incidence.
- Attributable mortality.
- Intervention effectiveness.
- Facility catchment coverage.
- Distance-to-care results.
- Area-specific scoring beyond extracted area-specific evidence.
- Claims based on WHO, WASH Cluster, or HDX values until export fields are verified.

## Audit rule

Every future empirical sentence must map to one of the following:

1. manual extraction row;
2. validated source inventory row;
3. peer-reviewed literature citation;
4. explicitly labeled methodological assumption.
