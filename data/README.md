# Data Directory

This directory documents data handling for the Gaza WASH public-health risk stratification project.

## Raw data

Raw public files should be stored locally under `data/raw/` and not committed unless licensing and file size are appropriate.

## Processed data

Processed analytic files should be reproducible from scripts and documented source downloads.

## Required metadata

Each downloaded file must record:

- source name;
- source URL;
- access date;
- publication or update date;
- geography;
- file format;
- license or reuse terms;
- extraction notes.

## Naming convention

Use lowercase snake case:

`source_geography_indicator_yyyy_mm_dd.ext`

Example:

`ocha_gaza_wash_sitrep_2026_05_15.pdf`
